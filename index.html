YSC Diffusion Stock

Solução em imagens de alta resolução gerada por IA

A YSC Diffusion Stock visa compartilhar seu amplo estoque de imagens geradas por IA para o público geral, além dos fluxos de trabalho (Worflows) usados para gerá-las. Utilizamos de modelos de difusão baseado nos modelos base do Stable Diffusion, uma IA generativa open-source criada pela Stability AI. A interface usada para que tais gerações fossem possíveis é a ComfyUI, onde os Workflows disponibilizados serão de uso exclusivo para tal interface.

A YSC Diffusion Stock possui um amplo catálogo de imagens geradas por IA, podendo ser conferidas na aba 'Imagens'. Aqui o usuário também está habilitado a enviar requisições de imagens diretamente para o servidor de geração, que necessita de uma verificação e geração manual, garantindo um resultado preciso e sem complicações.

Banco de imagens

Catálogo amplo com diversas imagens com alta resolução, além de diferentes tipos, estilo artistico, entre outros. Há um foco maior em imagens realistas, porém estilos como pintura à óleo, desenhos, vetores e animes também podem ser encontrados. As imagens são exibidas na página como um preview de baixa resolução, sendo a imagem disponível para download com resulução de até 4k.

Workflows

As imagens geradas usando modelos do Stable Diffusion foram geradas usando a interface ComfyUI, cujo funcionamento se baseia em nós interligados que executam os passos para gerar uma imagem. O conjunto interligado e funcional de tais nós se chamam workflows. Esta página disponibiliza diversos fluxos de trabalho para que você possa gerar suas próprias imagens de acordo com suas necessidades.

Requisições

Você poderá enviar um formulário para nós contendo requisições de imagens. Nesta página será possível detalhar o que você deseja e uma imagem será gerada conforme sua vontade. O tempo para receber as imagens pode variar conforme a disponibilidade de nossos colaboradores.
