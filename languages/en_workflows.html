Workflows

Basic Workflow

  • Download
  • Lora version
  • Capabilities: Basic image generation
  • Supports Flux: No
  • Custom node: None

This is the most basic workflow for ComfyUI. It uses only the standard interface nodes, without the need to install any custom nodes. Ideal for new users of both Stable Diffusion and ComfyUI itself.

Standard workflow + Hires-fix + Upscale

  • Download
  • Lora version
  • Capabilities: Image generation with focus on detail and resolution
  • Supports Flux: No
  • Custom node: None

This workflow combines standard image generation with the 'Highres-fix' process, which consists of upscaling the image and reprocessing it, increasing quality and detail. In addition, there is a final upscaling with the aim of making the resolution higher, ideal for projects.

Standard + Highres-fix + Ultimate Upscaler

  • Download
  • Lora version
  • Capabilities: Image generation with focus on detail and resolution
  • Supports Flux: No
  • Custom node: SD Ultimate Upscaler

This flow performs the same image generation and Highres-fix functions, however, in the final upscaling there is a custom node (SD Ultimate Upscaler) responsible for increasing the size of the images as many times as desired without compromising memory. Ideal for creating images in high resolutions (4k or higher).

Standard + ControlNet + Hires-fix + Ultimate Upscaler

  • Download
  • Lora version
  • Capabilities: Image generation with focus on detail and resolution
  • Supports Flux: No
  • Custom node: SD Ultimate Upscaler

This flow has a differential in using controlnet in the middle of the creation flow. Controlnet helps with the precision of prompts and details, allowing greater control during generation, in addition to increasing quality and efficiency. Hardware with good performance is recommended for higher creation speeds.
